# Shopping Cart Frontend

## Overview
The Shopping Cart Frontend is an Angular application designed to provide a user-friendly interface for managing a shopping cart. This application allows users to browse products, add items to their cart, view their cart's contents, and proceed to checkout.

## Features
- **Product List**: Displays a list of products available for purchase.
- **Shopping Cart**: Allows users to add, remove, and update quantities of items in the cart.
- **User Authentication**: Secure login functionality for user accounts.
- **Checkout Process**: Step-by-step process to complete a purchase.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/lipun619/ShoppingCartFrontend.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ShoppingCartFrontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   ng serve
   ```
5. Open your browser at `http://localhost:4200/`.

## Technologies Used
- Angular: A TypeScript-based open-source web application framework.
- HTML/CSS: For structuring and styling the application.
- Bootstrap: For responsive design and UI components.
- RxJS: For handling asynchronous data.
